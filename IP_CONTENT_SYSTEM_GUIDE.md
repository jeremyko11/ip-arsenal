## 专用Prompt使用指南

### Prompt 1: 原子化提取

**使用场景**：读完一章或一本书后
**输入**：书籍原文（划线部分）
**输出**：知识卡片（JSON格式）

**关键参数**：
- 思维模型：认知偏差/复利效应/马太效应/机会成本等
- 情感共鸣：焦虑/希望/愤怒/认同/惊讶/好奇
- 适用场景：短视频/文章/直播/课程/金句

**示例**：
```
输入：书籍原文关于"现状偏差"的段落

输出：
{
  "core_idea": "现状偏差让人困在不喜欢的工作中，机会成本比转行成本更高",
  "thinking_model": "认知偏差",
  "emotional_resonance": ["焦虑", "希望"],
  "applicable_scenarios": ["短视频", "文章"]
}
```

---

### Prompt 2: 内容转化

**使用场景**：有了知识卡片，要生成具体内容
**输入**：知识卡片 + 目标平台 + 受众痛点
**输出**：可直接发布的口播稿/文章

**关键技巧**：
- 开头钩子：痛点共鸣 / 反常识 / 悬念 / 数据冲击
- 故事案例：真实 > 书中 > 虚构
- 行动建议：具体、可执行、立即可做
- 结尾互动：提问 / 争议 / 承诺后续

**平台适配**：

| 平台 | 特点 | 内容结构 |
|------|------|----------|
| 抖音 | 3秒定生死 | 钩子+观点+金句 |
| 视频号 | 深度+温度 | 故事+观点+行动 |
| 公众号 | 逻辑严密 | 问题-分析-方案-总结 |
| 小红书 | 实用+颜值 | 清单体+emoji+干货 |

---

### Prompt 3: 跨书整合

**使用场景**：积累了多本书的卡片后
**输入**：多张知识卡片
**输出**：跨领域洞察报告

**价值**：
- 发现不同书籍的共同主题
- 建立跨领域的思维模型
- 策划系列内容（如"认知偏差10讲"）
- 强化IP定位的独特性

## 提高信息提取质量的秘诀

### 1. 带着问题读书

不要从头读到尾，而是：
- 先问自己：我想解决什么问题？
- 带着问题找答案
- 只提取与问题相关的内容

**你的核心问题清单**：
- 为什么聪明人也会做傻事？（认知偏差）
- 为什么有能力的人升不上去？（职场策略）
- 为什么明知道不合适还是放不下？（情感心理）
- 为什么谣言传播得比真相快？（社会现象）

### 2. 用自己的话重写（费曼技巧）

**不要**：复制原文
**要**：读完一段，合上书本，用自己的话讲出来

**检验标准**：
- 一个外行能听懂吗？
- 去掉专业术语了吗？
- 有具体的例子吗？

### 3. 标注情感共鸣点

内容传播的关键是**情绪**。

每提取一个观点，问自己：
- 这会让读者感到焦虑吗？（问题意识）
- 能给他们希望吗？（解决方案）
- 会引发愤怒吗？（不公现象）
- 能产生认同吗？（"我也是"）

### 4. 建立思维模型库

不要零散地记笔记，而是归入**思维模型**：

```
认知偏差/
├── 现状偏差
├── 幸存者偏差
├── 确认偏误
├── 沉没成本
└── 锚定效应

复利效应/
├── 财富复利
├── 知识复利
├── 关系复利
└── 健康复利
```

不同书籍的观点可以归入同一模型，形成深度洞察。

### 5. 输出倒逼输入

**不要**：读完10本书再开始输出
**要**：读完1章就尝试输出

**最小输出单位**：
- 一条朋友圈（100字）
- 一个短视频脚本（300字）
- 一篇短文（800字）

## 关键成功因素

### 1. 持续性 > 完美性

- 不要追求每条内容都完美
- 先完成，再完美
- 每天进步1%

### 2. 真实性 > 专业性

- 不要装专家，做"学习者的代言人"
- 分享你的真实困惑和成长
- "我也是这样"比"你应该这样"更有共鸣

### 3. 互动性 > 单向输出

- 每条内容都要引导互动
- 认真回复每一条评论
- 让粉丝参与内容创作

### 4. 差异化 > 全面性

- 不要什么话题都讲
- 找到你的独特视角（如"认知偏差解读职场"）
- 成为某个细分领域的专家

## 常见问题

### Q: 读一本书要花多长时间？

**A**: 不要追求读完，要追求**提取**。
- 一本书可能有100个观点
- 你只需要提取10个对你有用的
- 用1小时快速浏览，标记重点
- 用30分钟提取原子笔记

### Q: 如何避免内容同质化？

**A**:
- 加入**个人经历**和**故事**
- 用**跨学科视角**解读（心理学+经济学+生物学）
- 提出**反常识观点**
- 给出**可执行的行动建议**

### Q: 没有个人故事怎么办？

**A**:
- 用**朋友的经历**（征得同意）
- 用**历史故事**
- 用**新闻事件**
- 用**书籍中的案例**+你的解读

### Q: 如何保持创作灵感？

**A**:
- 建立**灵感收集库**（随时记录想法）
- 定期做**跨书整合**（发现新联系）
- 关注**评论区**（粉丝的问题就是选题）
- 阅读**不同领域**的书（跨界产生创新）
